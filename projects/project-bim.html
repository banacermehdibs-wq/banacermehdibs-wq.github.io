Revit Automation — Dynamo / Python

Year 2024–25
Role Script Development
Location Utrecht, NL
Tools Revit, Dynamo, CPython3

A custom CPython3 script for automatically placing NEN-standard dimension chains on floor plan views.

Manually dimensioning floor plans to NEN drafting standards is repetitive and error-prone — four parallel dimension chains per side, consistently offset, correctly filtered against curtain walls and diagonal geometry. This project automated that process inside Revit using Dynamo and CPython3.

Approach

The script reads wall and opening geometry directly from the Revit API, builds reference planes, and places four parallel dimension chains per elevation automatically. Key constraints were established through iterative debugging: no direct XYZ arithmetic on Revit's geometry objects, careful face-reference handling, and workarounds for grid-reference limitations in the API.

Outcome

The tool now runs across floor plan views at Bouwadvies Mur, cutting manual dimensioning time significantly on residential renovation and cadastral-split projects while keeping output consistent with NEN conventions.
